CSS Selectors

Core CSS Selectors

Е (Element)

Matches any E element (i.e., an element of type E). Example

Е F

Matches any F element that is a descendant of an E element. Example

E.classname

Matches the element E with the class attribute "classname". Example

E#id

Matches the element E with the id attribute "id". Example

E:active

The :active pseudo-class applies while an element is being activated by the user. For example, between the times the user presses the mouse button and releases it. Example

E:visited

The :link pseudo-class applies to links that have not yet been visited. The :visited pseudo-class applies once the link has been visited by the user. Example

E:first-line

Matches the element E with the id attribute "id". Example

E:first-letter

Matches the element E with the id attribute "id". Example

CSS 2.1 Selectors

*

Matches any element. Example

E > F

Matches any F element that is a child of an element E. Example

E + F

Matches any F element immediately preceded by a sibling element E. Example

E:first-child

Matches element E when E is the first child of its parent. Example

E:hover

Matches E when user hovers over E. Example

E[attr]

Matches any E element with the "attr" attribute set (whatever the value) Example

E[attr="name"]

Matches any E element whose "attr" attribute value is exactly equal to "name". Example

E[attr~="name"]

Matches any E element whose "attr" attribute value is a list of space-separated values, one of which is exactly equal to "name". Example

E:before E:after

Generated content before/after an E element. Example
