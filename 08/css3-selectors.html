CSS3 Selectors

E ~ F

Matches any F element preceded by an E element. Example

E[attr^="name"]

Matches an E element whose "attr" attribute value begins exactly with the string "name" Example

E[attr$="name"]

Matches an E element whose "attr" attribute value end exactly with the string "name" Example

E[attr*="name"]

Matches an E element whose "attr" attribute value contains the substring "name" Example

E[attr|="name"]

Matches any E element whose "attr" attribute value is a list of hyphen-separated values, the first being exactly equal to "name". Example

E:root

Matches an E element, root of the document - in HTML this is always the HTML-element.

E:last-child

Matches element E when E is the last child of its parent. Example

E:empty

Matches an E element, that has no children, including text nodes Example

E:target

Matches an E element being in the target of the referring URI Example

E:checked

h A user interface element E which is checked (for instance a radio-button or checkbox) Example

E:selection

Matches the portion of an E element that is currently selected/highlighted by the user. Example

E:enabled / E:disabled

A user interface element E which is enabled or disabled (for instance a radio-button or checkbox) Example

E:not(s)

Matches an E element that does not match simple selector s Example

E:nth-child

Matches an E element, the n-th child of its parent. Example
